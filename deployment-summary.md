# Azure Resource Deployment Summary

## Selected Region

I deployed my Azure resource in **West Europe** because it was the available region during resource creation. West Europe is one of Microsoft's major Azure regions and provides broad support for Azure services, making it a reliable choice for deploying cloud resources.

## Shared Responsibility Model

The Azure Shared Responsibility Model means that security is shared between Microsoft and the customer.

- **Microsoft's responsibility:** Microsoft is responsible for securing the underlying cloud infrastructure, including the physical data centers, networking, hardware, and the Azure platform.
- **My responsibility:** I am responsible for securing the Azure resource I deployed by managing access permissions, protecting my data, configuring security settings appropriately, and maintaining the security of my applications or virtual machines where applicable.

This model ensures that while Microsoft secures the cloud infrastructure, I remain responsible for securing my resources and the data stored within them.
